Heap (куча) используется для выделения памяти под объекты и классы.
-   У всех потоков имеется глобальный доступ к куче — любой поток может сослаться на любой размещенный в ней объект.
-   Куча создаётся в момент запуска приложения и уничтожается вместе с его завершением.
-   Размер оперативной памяти, выделенный под кучу, может меняться во время жизни приложения как в сторону увеличения, так и в сторону уменьшения.
Стековая память используется для хранения стеков исполнения потоков. Каждый такой стек работает по схеме LIFO.
-   При создании потока происходит создание его стека исполнения.
-   По мере исполнения потоком программы перед каждым вызовом метода в стеке создается новый элемент (stack frame, стек-фрейм), который используется для хранения локальных переменных примитивных типов и ссылок на объекты из кучи.
-   После завершения исполнения метода стек-фрейм удаляется и тем самым ранее занимаемая им память может быть использована под следующие вызовы.
-   При завершении потока происходит удаление его стека.
