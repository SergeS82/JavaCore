-   Alt + 1             - свернуть / развернуть вкладку Project
-   Ctrl + N            - поиск по библиотекам
-   Ctrl + Shift + F    - поиск по проекту
-   Ctrl + E            - список последних файлов
-   Ctrl + O            - список всех методов которые можно переопределить
-   Ctrl + Alt + V      - автоматически создать опредление переменной после написания например new ArrayList<String>();
                          или например LocalDate.now();
-   Ctrl + Alt + M      - вытащить выделенный код или строку кода в отдельный метод
-   Ctrl + D            - дублировать строку или блок
-   Ctrl + Y            - удалить строку
-   Ctrl + W            - расширить выделение
-   Ctrl + Shift + W    - сузить выделение
-   Ctrl + P            - вызвать подсказку принудительно
-   Ctrl + Q            - краткая документация
-   Ctrl + F12          - оглавление класса
-   Ctrl + Home         - в начало класса
-   Ctrl + End          - в конец класса
-   Ctrl + вверх/вниз   - пролистывание
-   Alt + вверх/вниз    - перемещение по блокам кода
-   Ctrl+Alt+лево/право     - перемещение по предыдущим местам просмотра/правок
-   Ctrl+Shift+вверх/вниз   - перемещение строки/блока вверх/вниз
-   Ctrl + Alt + O      - убрать лишний импорт
-   Ctrl + Alt + L      - автоформатирование
